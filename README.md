# elasticguard-ai

## 🚀 Quick Start with GitHub Codespaces

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/kumkumchoudhary14/elasticguard-ai)

1. Click the button above to open in Codespaces
2. Run `docker-compose up --build` in the terminal
3. Open the forwarded port 8000 to see the dashboard
4. In a new terminal, run `python iot_simulator.py` to generate data
